Django
	settings.py
		Cuando creas una aplicación tienes que añadirla en INSTALLED_APPS (variable del fichero settings.py).
			Le pones el nombre tal cual, sin nada delante
		Puedes añadir una variable STATIC_URL que define la dirección en el url donde se encuentran
			Luego puedes modificar el archivo html correspondiente poniendo { % load static % }
	urls.py (de la aplicación)
		El primer argumento de path es lo que habría escrito en: http://localhost:8000/¡aquí!
			Para que te acepte variables:
			path("<¡tipo de variable!:¡nombre de la variable!>/", ¡función a la que llamar!, name=¡no importa!)
			Y en urls.py (del proyecto)
			path('', include(¡nombre aplicación!.urls))
